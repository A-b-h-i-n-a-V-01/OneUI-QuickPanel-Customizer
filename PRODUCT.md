# Product

<!-- impeccable:product-schema 1 -->

## Platform

web

## Users
Samsung power users, theme creators, and Android customization enthusiasts who want to create pixel-perfect custom backgrounds for their Galaxy device Quick Panels using Theme Park/QuickStar.

## Product Purpose
To allow users to manually calibrate Quick Panel screenshot regions and crop/position/edit wallpapers exactly behind those regions, exporting precise PNG overlays for Theme Park.

## Positioning
A manual, high-precision visual crop and transform tool that lets creators bypass unreliable auto-detection to get pixel-accurate alignment on any screen resolution.

## Operating Context
Customizers upload a screenshot of their quick panel, calibrate the box coords visually, overlay a wallpaper image, scale/rotate/blur it, preview it in a live mobile mockup, and export.

## Capabilities and Constraints
- Screenshot uploading and manual bounding box calibration.
- Wallpaper scaling, panning, rotation, and opacity/blur filtering.
- Dual-image overlay canvas rendering via Konva.
- Local storage saving for calibration profiles.
- Desktop and mobile-responsive viewport.

## Brand Commitments
- Name: DIY OneUI (Quick Panel Editor)
- Identity: Sleek, high-contrast dark mode matching modern Samsung galaxy aesthetics.

## Evidence on Hand
- Full React + Vite codebase with tailwindcss styling and konva/framer-motion.

## Product Principles
- Pixel-perfect manual control over automated assumptions.
- Performance-first client-side canvas transformations.
- Clean, dark-themed workspaces that respect the visual context of the themes.
